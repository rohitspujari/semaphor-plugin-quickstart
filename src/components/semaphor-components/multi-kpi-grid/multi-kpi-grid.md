# Multi KPI Grid

A multi-input KPI grid example that uses `tabMetadata` and `cardMetadata` to render KPI tiles.
It demonstrates how to parse KPI data, format values, and show comparison deltas with the helper utilities.

## Props

Uses `MultiInputVisualProps`.

## Notes

- Expects each tab to be configured as a KPI card.
- Uses `cardMetadata` for comparison metadata and number formatting.
